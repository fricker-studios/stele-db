# Stele brand bundle

**Mark:** Strata · **Wordmark:** Steledb · **Accent:** Lapis `#2f6bd6` · **Type:** Space Grotesk

## svg/ — scalable sources
- `stele-mark.svg` / `stele-mark-reversed.svg` — the mark, for light / dark backgrounds
- `stele-app-icon-lapis.svg` / `-ink.svg` / `-paper.svg` — rounded app-icon tiles
- `stele-wordmark.svg` / `-reversed.svg` — full lockup *(text uses Space Grotesk via webfont; renders
  correctly when the SVG is opened directly or placed inline. For `<img>` embedding or anywhere the
  font may be missing, use the wordmark PNGs below — they have the type baked in.)*

## png/ — rasterized exports (transparent)
- `stele-mark-256/512/1024.png`, `stele-mark-reversed-512.png`
- `favicon-16/32/48.png`, `apple-touch-icon-180.png`, `icon-512/1024.png` (Lapis tile)
- `icon-ink-512/1024.png`
- `stele-wordmark.png` / `stele-wordmark-reversed.png` — type baked in, embed-safe

## palette
| | hex | use |
|---|---|---|
| Paper | `#fbfbfa` | page background |
| Mist | `#f3f4f2` | panels / alt background |
| Line | `#e7e8e6` | hairlines |
| Slate | `#5c626b` | secondary text |
| Ink | `#15171c` | primary text |
| **Lapis** | **`#2f6bd6`** | the single accent — links, primary button, mark, code keywords |

Type: **Space Grotesk** (display + UI), **Hanken Grotesk** (body), **JetBrains Mono** (code).
